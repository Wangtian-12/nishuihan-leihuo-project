逆水寒｜网易雷火
本项目为个人整理的《逆水寒》游戏全维度资料。


一、游戏背景

《逆水寒》（端游）的故事背景锚定在北宋末年，彼时朝堂腐败、奸臣当道，方腊起义席卷江南，金辽两国虎视眈眈，内忧外患交织之下，江湖势力也借机崛起，形成了“乱世藏锋芒，江湖定乾坤”的核心基调。其世界观基于温瑞安《四大名捕》及《说英雄谁是英雄》系列小说构建而成，包含该武侠世界中的众多高手与大侠，如诸葛神侯与四大名捕等核心人物。
游戏核心线索围绕“逆水寒剑”展开——这把蕴含上古神力的名剑，传说能逆转乾坤，却也自带戾气，持有者轻则心性大乱，重则祸及天下。正因如此，江湖各大门派（其中三清山为自在门所在地，是玩家初始成长的场景，另有白帝城、绝情谷等）、朝廷禁军、民间义军，皆为争夺此剑、掌控乱世走向而明争暗斗，恩怨纠缠。
玩家自幼被师父叶哀禅带上三清山收养，在这个世外桃源习武成长，成年后踏入江湖，以一名身世神秘的江湖弟子身份，意外卷入这场纷争之中。从初入江湖、懵懂无知的无名小卒，逐步揭开自己的身世之谜，在“家国大义”与“江湖恩怨”、“个人情仇”与“天下苍生”之间做出抉择，最终成长为影响乱世格局的关键人物。
游戏深度还原了北宋的历史风貌，从汴京的繁华市井、杭州的烟雨江南，到江南水乡的小桥流水、塞北边疆的苍茫辽阔，每一处场景都极具古风韵味。同时，游戏打造了“会呼吸的江湖”体验，NPC拥有独立的生活轨迹、情绪变化，会根据玩家的行为做出不同反应，让整个江湖显得真实可触、充满烟火气，这也是《逆水寒》系列的核心特色之一。

二、创作团队

《逆水寒》（端游）的核心创作与研发团队是网易雷火工作室，该工作室成立于2007年，是网易旗下专注于高品质游戏研发的核心力量，总部位于杭州，汇聚了上千名专业研发人才，涵盖游戏策划、美术设计、程序开发、剧情创作、运营推广等多个领域，是国内游戏行业极具影响力的研发团队之一。
为打造《逆水寒》这款旗舰武侠作品，雷火工作室组建了专属核心研发小组，集结了网易内部顶尖的武侠游戏研发人才，同时邀请国内知名历史学者、武侠作家参与背景设定与剧情打磨，既保证了游戏的历史底蕴，又兼顾了武侠故事的趣味性与沉浸感，其剧情设定深度贴合温瑞安武侠小说的核心调性。
作为网易的核心工作室，雷火不仅深耕武侠游戏领域，还在竞技游戏、仙侠游戏等多个赛道布局，代表作包括《永劫无间》《倩女幽魂》等多款爆款产品，凭借“高品质、高沉浸、强创新”的研发理念，成为国内游戏行业的标杆团队，其中《逆水寒》与《永劫无间》是雷火工作室的两大核心旗舰产品。
《逆水寒》端游于2018年正式上线，手游则于2023年6月30日开启公测。该系列作品自上线以来能够持续保持热度，离不开雷火工作室的持续投入与优化——研发团队始终关注玩家需求，定期推出新剧情、新玩法、新职业，优化游戏画质与体验，让这款作品不断迭代升级，成为国内武侠MMORPG的标杆之作。

三、游戏玩法

《逆水寒》（含端游、手游）作为一款3D武侠MMORPG，带有开放世界元素，玩法覆盖剧情体验、职业养成、竞技对抗、休闲娱乐、社交互动等多个维度，兼顾硬核玩家与休闲玩家的需求，核心玩法贴合武侠设定，沉浸感与趣味性兼具，具体分为以下几类，均基于官方公开玩法与权威资料整理：

（一）剧情与奇遇玩法
1.  主线剧情：以“逆水寒剑”为核心线索，串联起北宋末年的乱世纷争，基于温瑞安武侠小说世界观展开，采用电影级叙事手法，搭配专业配音与细腻剧情动画，让玩家沉浸式体验江湖恩怨与家国大义，剧情分支丰富，玩家的选择会影响部分剧情走向与NPC态度。
2.  支线与奇遇：游戏内拥有上百个支线任务与隐藏奇遇，玩家在江湖行走时，可能因捡起一件物品、与某个NPC对话、触发特定场景而解锁奇遇，奇遇无固定流程，奖励丰富，部分奇遇可解锁隐藏剧情、稀有道具或专属外观，极大提升了游戏的探索性，这也是《逆水寒》“会呼吸的江湖”特色的重要体现。
（二）职业养成玩法
1.  职业选择：目前游戏内有8大核心职业，分别是碎梦（刺客，高爆发近战）、血河（战士，长枪群体输出）、神相（法师，远程元素伤害）、素问（辅助，治疗续航）、铁衣（坦克，承伤控场）、龙吟（近战，机动输出）、玄机（远程，精准打击）、鸿音（辅助，增益控场），覆盖全职业定位，玩家可根据自身喜好选择，每个职业均有明确的官方设定与技能体系。
2.  技能与经脉：每个职业共有8个武器技能和12个职业技能，玩家可通过升级、佩戴技能书调整技能搭配，适配不同玩法场景；同时可通过修炼经脉、提升修为，增强自身基础属性，解锁职业专属特性，战斗界面可自定义技能栏，灵活搭配武学技能与职业技能。
3.  装备与锻造：游戏内装备分为不同品质，每个装备带有专属词条，玩家可通过副本掉落、任务获取、锻造打造等方式获取装备，再通过洗练、强化、镶嵌等玩法，提升装备属性，打造个性化装备套装，其中独珍装备是游戏内的稀有装备类型，可通过探索、副本、竞技等玩法获取。
（三）PVE与PVP竞技玩法
1.  PVE副本：分为日常副本与团本挑战，其中日常副本包括武林风云录、万象楼等，团本则有逆水寒、昆仑墟遗迹、风雪铁牢关、仓山蛊影、南海乱墟等，其中南海乱墟为12人海洋团本，蝶梦三折为戏曲主题6人副本，日常副本难度适中，适合玩家日常刷取材料、经验与装备；团本需要多人组队协作，分工明确（坦克承伤、输出打伤害、辅助治疗），考验团队配合能力，通关可获得稀有装备、外观等奖励。
2.  PVP竞技：包括1V1/3V3竞技场、跨服战场（试剑天下、卧龙岭、怒海等）、帮会战等，其中试剑天下依靠击杀和助攻获取分数，卧龙岭为占点类战场，帮会战为五团制对抗模式，竞技场考验个人操作与技能搭配，战场与帮会战考验团队战术与协作能力，玩家可通过竞技获得积分与奖励，满足竞技对抗需求，此外还有朔夜猎影等黑暗夺宝竞技玩法。
（四）休闲与社交玩法
1.  庄园系统：玩家可拥有专属庄园，通过种植、养殖、建造、装饰等方式，打造属于自己的古风家园，可邀请好友参观、互动，解锁休闲任务与专属奖励，部分版本还会推出庄园组件奖励，如南海乱墟主题庄园组件。
2.  生活技能：涵盖钓鱼、烹饪、采药、挖矿、锻造、裁缝等，玩家可学习不同生活技能，制作食物、药品、装备等，既能满足自身游戏需求，也可通过游戏内交易获得游戏货币，是休闲玩家的核心玩法之一。
3.  社交系统：拥有帮派、师徒、好友、姻缘等完善的社交体系，玩家可加入帮派参与帮派活动、加入师徒关系学习玩法技巧、与好友组队冒险、结亲组建游戏伴侣关系，增强游戏的社交属性与归属感，其中帮会战是帮派社交的核心玩法之一。
（五）赛季制玩法
游戏采用赛季制更新模式，每个赛季会推出新剧情、新玩法、新装备与平衡调整，降低新玩家入门门槛，同时为老玩家提供新鲜体验，确保游戏的长期活力，如2025年4月版本更新中，就包含新赛季战令、新副本、新装备等内容。

四、面向群体

《逆水寒》的面向群体以喜爱武侠文化、追求沉浸式游戏体验的玩家为主，同时覆盖不同年龄段、不同玩法需求的玩家，结合官方数据与玩家画像调查，具体可分为以下几类：
1.  武侠文化爱好者：这类玩家热衷于古风武侠题材，喜欢沉浸式的江湖剧情、武侠设定与古风场景，《逆水寒》基于温瑞安武侠小说的背景设定、丰富的剧情脉络与贴合武侠的玩法，能够精准满足这类玩家的需求，是游戏的核心受众之一。
2.  休闲娱乐玩家：这类玩家占比最高，约69%，被称为“乐子人”，不追求高强度竞技与养成，主要热衷于游戏内的休闲玩法，如庄园建造、生活技能、游戏内拍片、创意工坊场景搭建等，享受游戏带来的轻松体验，游戏“殊途同归”的养成模式的也降低了这类玩家的入门门槛。
3.  轻度养成玩家：这类玩家约占30%，被称为“咸鱼”玩家，不刻意“肝”游戏进度，偶尔参与副本、任务与社交活动，无需高强度投入即可跟上游戏节奏，游戏不将氪金与数值挂钩的设定，也契合这类玩家的需求。
4.  硬核竞技玩家：这类玩家占比约1%，被称为“肝帝”，热衷于PVP竞技、高难度团本挑战，追求职业养成的极致与操作技巧的提升，游戏丰富的竞技玩法与高难度副本能够满足这类玩家的需求。
5.  全年龄段玩家：游戏打破年龄壁垒，用户数量已突破一个亿，其中包含超过300万中老年玩家，这类玩家主要热衷于游戏内的休闲玩法，如麻将、航海等，形成了“全家共玩”的场景，覆盖不同年龄段的玩家需求。
